Sales Order Amendment

After confirmation of a sales order, Customer may wish to change the ordered quantity or negotiation with price or Payment terms or general terms & conditions. At the time we need to amend the sales order and also user needs to have a track on the amendment

Step 1:

Click on amendment button which appears on the sales order stage.

Step 2:

Once amendment is done, now you are able to see a new button called Set to quotation click on the button to make necessary changes on the order.

Step 3:

Now you can see the order is moved to quotation state and it is editable, to know the which user done the changes view the logs.

Step 4:

Now you can see the order is moved to quotation state and it is editable, to know the which user done the changes view the logs.

Step 5:

To view the previous order click on the list, a wizard will appear on the screen there user can view the order lines with total amount and taxes.
